How It Works

1

EPA Ratings

The model calculates Expected Points Added (EPA) for each team based on offensive and defensive performance throughout the season.

2

QB Adjustments

Quarterback changes significantly impact spreads. The model adjusts predictions based on who's actually starting each week.

3

Edge Calculation

Compares the model's predicted spread to Vegas lines. Larger edges (Tier 1) indicate higher confidence picks.
